# IELTS Interactive HTMLs — AI Correction & TTS — Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a parallel `Interactive/` set of 40 IELTS lesson HTMLs with an inline AI correction feature (red-pen markup over the student's typed draft), browser TTS for model answers and clicked words, draft persistence, and a free-tier Aliyun Function Compute backend calling Zhipu. Host the HTMLs in a new public Aliyun Supabase Storage bucket.

**Architecture:** Three independent components. (A) Aliyun Function Compute Node 18 endpoint at `*.fcapp.run` validates a 50–150 word draft, rate-limits in-memory, calls Zhipu's latest free GLM model with a minimum-correction system prompt, returns bilingual errors. (B) A re-runnable Python script clones each original lesson HTML, embeds Caveat + Indie Flower as base64 woff2 in a `@font-face`, hides the original `.lines` divs in `.draft-page`, adds overlay textarea/markup/output elements with their own scrolling lined background, and injects a self-contained `<script>` block carrying all browser JS. (C) Aliyun Supabase Storage public bucket hosts the 40 generated HTMLs and one shared `pronunciations.json` for IPA tooltips.

**Tech Stack:**
- Function Compute: Node.js 18 (built-in `fetch`), Serverless Devs (`s deploy`)
- Build script: Python 3.11 (`pathlib`, `re`, `base64`, `argparse`)
- Browser JS: vanilla ES2020 (no framework, no bundler), Web Speech API, `localStorage`, `sessionStorage`
- Fonts: Caveat 400 + Indie Flower 400 (woff2, SIL OFL)
- Hosting: Aliyun-hosted Supabase Storage public bucket via the `@aliyun-rds/supabase-mcp-server` MCP
- LLM: Zhipu open.bigmodel.cn `/api/paas/v4/chat/completions`, latest free GLM model

**Spec reference:** [docs/superpowers/specs/2026-04-25-ielts-ai-correction-design.md](../specs/2026-04-25-ielts-ai-correction-design.md). Every task references the relevant spec section.

```js
const CORS_HEADERS = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Methods': 'POST, OPTIONS, GET',
  'Access-Control-Allow-Headers': 'Content-Type',
  'Content-Type': 'application/json'
};

function respond(statusCode, body) {
  return {
    statusCode,
    headers: { ...CORS_HEADERS },
    body: typeof body === 'string' ? body : JSON.stringify(body)
  };
}

export async function handler(event) {
  const method = (event.httpMethod || event.method || '').toUpperCase();
  const path = event.path || event.rawPath || '/';

  if (method === 'OPTIONS') {
    return { statusCode: 204, headers: { ...CORS_HEADERS }, body: '' };
  }

  if (method === 'GET' && path === '/health') {
    return respond(200, { ok: true });
  }

  if (method !== 'POST') {
    return respond(405, { error: '方法不允许 / Method not allowed.' });
  }

  // POST handling lives in subsequent tasks
  return respond(501, { error: '未实现 / Not implemented yet.' });
}

// Aliyun FC HTTP-trigger style fallback in case of `event.handler`-style invocation
export default handler;
```

```js
// In-memory rate limiter — Map<ip, { count, windowStart }>.
// Resets when the FC instance recycles, which is acceptable per spec §5.4.
const RATE_LIMIT = new Map();
const RATE_LIMIT_MAX = 30;
const RATE_LIMIT_WINDOW_MS = 60 * 60 * 1000;

export function __resetRateLimit() {
  RATE_LIMIT.clear();
}

function getClientIP(event) {
  const xff = event.headers?.['x-forwarded-for'] || event.headers?.['X-Forwarded-For'];
  if (xff) return xff.split(',')[0].trim();
  return event.clientIP || event.requestContext?.identity?.sourceIp || 'unknown';
}

function checkRateLimit(ip) {
  const now = Date.now();
  const entry = RATE_LIMIT.get(ip);
  if (!entry || now - entry.windowStart > RATE_LIMIT_WINDOW_MS) {
    RATE_LIMIT.set(ip, { count: 1, windowStart: now });
    return { ok: true };
  }
  if (entry.count >= RATE_LIMIT_MAX) {
    return { ok: false };
  }
  entry.count += 1;
  return { ok: true };
}
```

```js
const ZHIPU_MODEL_ID = process.env.ZHIPU_MODEL_ID || 'glm-4.7-flash';
const ZHIPU_URL = 'https://open.bigmodel.cn/api/paas/v4/chat/completions';
const SYSTEM_PROMPT = `You are a careful English teacher correcting a 14-16 year old Chinese student's
short written answer (50-150 words) for an IELTS speaking lesson.

Make the MINIMUM changes needed for the writing to be grammatically correct and
to use words correctly. Your job is correction, not enhancement.

Fix:
- Grammar errors (verb tense, articles, subject-verb agreement, prepositions,
  word order, plurals, capitalization, punctuation)
- Spelling
- Wrong word choice ONLY when a word is genuinely incorrect (mistranslation,
  wrong sense, non-existent word)

Do NOT:
- Replace words that are already correct, even if simple or basic
- Add new ideas, examples, details, opinions, or sentences not in the student's draft
- Delete the student's ideas
- Restructure sentences unless grammar requires it
- Change length by more than 10 words from the student's original

Return ONLY the corrected text as plain prose. No preamble, no markdown,
no commentary, no quotes, no bullet points.`;

// Injectable for tests
let _zhipuFetcher = (url, opts) => fetch(url, opts);
export function __setZhipuFetcher(fn) { _zhipuFetcher = fn; }
export function __resetZhipuFetcher() { _zhipuFetcher = (url, opts) => fetch(url, opts); }

function isZhipuQuotaError(jsonOrText) {
  if (!jsonOrText) return false;
  const code = jsonOrText.error?.code;
  if (code === 1113 || code === 1301) return true;
  const msg = (jsonOrText.error?.message || '').toLowerCase();
  return msg.includes('quota') || msg.includes('limit') || msg.includes('余额');
}

async function callZhipu(draft) {
  const apiKey = process.env.ZHIPU_API_KEY;
  if (!apiKey) {
    return { ok: false, status: 503, error: 'AI 服务暂时不可用 / AI service temporarily unavailable.' };
  }
  let upstream;
  try {
    upstream = await _zhipuFetcher(ZHIPU_URL, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${apiKey}`
      },
      body: JSON.stringify({
        model: ZHIPU_MODEL_ID,
        temperature: 0.3,
        max_tokens: 500,
        messages: [
          { role: 'system', content: SYSTEM_PROMPT },
          { role: 'user', content: draft }
        ]
      })
    });
  } catch (e) {
    return { ok: false, status: 503, error: 'AI 服务暂时不可用 / AI service temporarily unavailable.' };
  }

  let parsed;
  try { parsed = await upstream.json(); } catch { parsed = null; }

  if (!upstream.ok) {
    if (isZhipuQuotaError(parsed)) {
      return { ok: false, status: 503, error: 'AI 今日额度已用完，请明天再试 / AI quota exhausted today. Try again tomorrow.' };
    }
    return { ok: false, status: 503, error: 'AI 服务出错，请稍后再试 / AI service error. Please try later.' };
  }

  const content = parsed?.choices?.[0]?.message?.content;
  if (typeof content !== 'string' || !content.trim()) {
    return { ok: false, status: 500, error: 'AI 返回了意外的响应 / AI returned an unexpected response.' };
  }
  return { ok: true, corrected: content.trim() };
}
```

```js
// Pure functions; reused inline at build time inside inserted_script.js.

export function wordDiff(a, b) {
  const aw = a.split(/\s+/).filter(Boolean);
  const bw = b.split(/\s+/).filter(Boolean);
  const m = aw.length, n = bw.length;
  // LCS DP
  const dp = Array.from({length: m+1}, () => new Int32Array(n+1));
  for (let i = m-1; i >= 0; i--) {
    for (let j = n-1; j >= 0; j--) {
      if (aw[i] === bw[j]) dp[i][j] = dp[i+1][j+1] + 1;
      else dp[i][j] = Math.max(dp[i+1][j], dp[i][j+1]);
    }
  }
  const out = [];
  let i = 0, j = 0;
  while (i < m && j < n) {
    if (aw[i] === bw[j]) { out.push({op:'keep', word: aw[i]}); i++; j++; }
    else if (dp[i+1][j] >= dp[i][j+1]) { out.push({op:'delete', word: aw[i]}); i++; }
    else { out.push({op:'insert', word: bw[j]}); j++; }
  }
  while (i < m) { out.push({op:'delete', word: aw[i++]}); }
  while (j < n) { out.push({op:'insert', word: bw[j++]}); }
  return out;
}

export function classifyPairs(segs) {
  const out = [];
  let i = 0;
  while (i < segs.length) {
    const cur = segs[i];
    const next = segs[i+1];
    if (cur.op === 'delete' && next?.op === 'insert') {
      const x = cur.word, y = next.word;
      if (y.startsWith(x) && y !== x) {
        out.push({op:'suffix-add', kept: x, added: y.slice(x.length)});
      } else if (y.endsWith(x) && y !== x) {
        out.push({op:'prefix-add', added: y.slice(0, y.length - x.length), kept: x});
      } else {
        out.push({op:'replace', deleted: x, inserted: y});
      }
      i += 2;
    } else {
      out.push(cur);
      i += 1;
    }
  }
  return out;
}

function esc(s) {
  return String(s).replace(/&/g,'&amp;').replace(/</g,'&lt;').replace(/>/g,'&gt;');
}

export function renderMarkup(classified) {
  const parts = [];
  classified.forEach((seg, idx) => {
    const space = idx > 0 ? ' ' : '';
    switch (seg.op) {
      case 'keep':
        parts.push(`${space}${esc(seg.word)}`);
        break;
      case 'delete':
        parts.push(`${space}<del class="del">${esc(seg.word)}</del>`);
        break;
      case 'insert':
        parts.push(`${space}<span class="gap-anchor"><ins class="ins-above">${esc(seg.word)}</ins></span>`);
        break;
      case 'replace':
        parts.push(`${space}<span class="replace-pair"><del class="del">${esc(seg.deleted)}</del><ins class="ins-above">${esc(seg.inserted)}</ins></span>`);
        break;
      case 'suffix-add':
        parts.push(`${space}${esc(seg.kept)}<ins class="ins-suffix">${esc(seg.added)}</ins>`);
        break;
      case 'prefix-add':
        parts.push(`${space}<ins class="ins-prefix">${esc(seg.added)}</ins>${esc(seg.kept)}`);
        break;
    }
  });
  return parts.join('');
}
```

```python
import base64
import re
import sys
from pathlib import Path
from typing import Iterable

TEMPLATE_DIR = SCRIPT_DIR / "templates"
FONT_DIR = SCRIPT_DIR / "fonts"


def _files_to_process(in_path: Path) -> Iterable[Path]:
    if in_path.is_file():
        if re.match(r"Week_\d+_Lesson_Plan\.html$", in_path.name):
            yield in_path
    else:
        for p in sorted(in_path.glob("Week_*_Lesson_Plan.html")):

def main() -> int:
```

```python
#!/usr/bin/env python3
"""Extract bolded vocabulary from all 40 lesson HTMLs, look up IPA via CMU dict,
write a single pronunciations.json mapping.

Output is a flat JSON: { "word": "/ɪpə/" }.
"""
from __future__ import annotations
import json
import re
import sys
import urllib.request
from pathlib import Path

REPO = Path(__file__).resolve().parents[1]

CMU_IPA_URL = "https://raw.githubusercontent.com/menelik3/cmudict-ipa/master/cmudict-0.7b-ipa.txt"


def extract_words() -> set[str]:
    """Find every <strong>...</strong> within .vocab-table and .model-box across all 40 originals."""
    words: set[str] = set()
    bold = re.compile(r"<strong[^>]*>([^<]+)</strong>", re.IGNORECASE)
    for p in sorted(REPO.glob("Week_*_Lesson_Plan.html")):
        text = p.read_text(encoding="utf-8")
        for m in bold.finditer(text):
            content = m.group(1).strip()
            for token in re.findall(r"[A-Za-z][A-Za-z'-]*", content):
                if 2 <= len(token) <= 30:
                    words.add(token.lower())
    return words


def fetch_cmu_ipa() -> dict[str, str]:
    """CMU IPA dict format: WORD<tab>/aɪ p ə/"""
    req = urllib.request.Request(CMU_IPA_URL, headers={"User-Agent": "ielts-interactive/1.0"})
    with urllib.request.urlopen(req, timeout=60) as resp:
        text = resp.read().decode("utf-8", errors="replace")
    out: dict[str, str] = {}
    for line in text.splitlines():
        line = line.strip()
        if not line or line.startswith(";;;"):
            continue
        parts = line.split("\t")
        if len(parts) != 2:
            continue
        word, ipa = parts[0].lower(), parts[1].strip()
        # Drop multi-pronunciation suffixes like "address(1)"
        word = re.sub(r"\(\d+\)$", "", word)
        # Take first encountered pronunciation only.
        out.setdefault(word, ipa)
    return out


def main() -> int:
    words = extract_words()
    print(f"Extracted {len(words)} unique words from lesson HTMLs.")
    print("Fetching CMU IPA dict...")
    cmu = fetch_cmu_ipa()
    print(f"CMU dict has {len(cmu)} entries.")
    out = {w: cmu[w] for w in words if w in cmu}
    print(f"Matched {len(out)} / {len(words)}.")
    out_path = REPO / "pronunciations.json"
    out_path.write_text(json.dumps(out, ensure_ascii=False, sort_keys=True, separators=(',', ':')), encoding="utf-8")
    size = out_path.stat().st_size
    print(f"Wrote {out_path} ({size:,} bytes).")
    if size > 1_048_576:
        print("WARNING: pronunciations.json exceeds 1 MB.", file=sys.stderr)
    return 0


if __name__ == "__main__":
    sys.exit(main())
```
